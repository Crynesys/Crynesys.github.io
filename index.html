nOSChat

The little chat where you can send asset with the help of nOS funtionalities

nOS functionalities accessible only accessing the site by nOS Client

  • Enter in the nOS Chat (it takes a bit of time..)
Source Repository

Register

When you access for the first time please register a new user. If done in the nOS Client the NEO address will be associated with the new user.

Search

Search for groups or users in the chat

Results

In the results the groups or users with nOS functionalities enabled are tagged with a "nOS" label

Join a group

Join a group with nOS Functionalities.

Send to a group

Using the button on the upper right corner, open the group tab. There you will find infos about the group and the form to send NEO or Gas to the address associated.

Assets

Select the asset to send.

Create a group

Create a group entering name, description and if done in the nOS Client the address associated ( !!can not be added later !!)

Other User with nOS Client

The search function return groups and users. If a user have an address associated the result will be tagged.

Send to User

Selecting a User you will see an information dialog with the option to send asset to the user's address.

Asset sent successfully

Well everything is gone right and your asset is on the way to the other address...If something goes wrong you will see the error returned from nOS but is ugly and do not wanto to talk about it.
